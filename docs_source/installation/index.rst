For advanced users
==================

Create a new Python (3.7) environment with :code:`conda` or :code:`mamba`:

.. code-block:: bash

	mamba create -n <NAME> python=3.7  # <NAME> == chosen name, e.g. "pp-env"	


Install phenopype with :code:`pip install phenopype`. 

.. code-block:: bash

	mamba activate <NAME>  			   
	pip install phenopype

To work interactively, I strongly recommend to use `VScode <https://code.visualstudio.com/>`_ or `Spyder <https://docs.spyder-ide.org/current/index.html>`_.
